## ðŸš€ Quick Start Guide

1. **Run Setup Wizard**
   ```bash
   python3 run_setup_wizard.py
   ```

2. **Launch GUI**
   ```bash
   python3 liberator_gui.py
   ```

3. **Extract a Project**
   - Go to Extract tab
   - Select project
   - Click "Liberate Project"

4. **Use AI Assistant**
   - Go to AI Assistant tab
   - Ask questions or repair code

5. **Validate Compatibility**
   - Go to Executive Producer tab
   - Select project and platforms
   - Click "Validate Compatibility"
   - Click "Auto-Fix Issues"

6. **Track Projects**
   - Go to Projects tab
   - View all your projects
   - Track issues and fixes
